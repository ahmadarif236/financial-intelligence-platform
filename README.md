# GCC CFO AI - Financial Intelligence Platform

This is the backend service for the GCC CFO AI platform, built with FastAPI and PostgreSQL (Supabase).

## Deployment
This repository is automatically synced to Hugging Face Spaces.

### Requirements
- Docker
- Python 3.10+
- Supabase (PostgreSQL)
- Cerebras Cloud API Key
